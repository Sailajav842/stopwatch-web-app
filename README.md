# Stopwatch Web App ⏱️

A responsive stopwatch application developed using HTML, CSS, and JavaScript.

## Features
- Start Timer
- Pause Timer
- Reset Timer
- Lap Time Recording
- Responsive Design
- Modern User Interface

## Technologies Used
- HTML5
- CSS3
- JavaScript
